*****les commande de base *******
.telecharger image

        docker pull images

.Creer un conteneur

         docker run --name [name conteneur] [image]

.lister les conteneurs actifs et tous les conteneurs:
        
         docker ps (actif)
         docker ps -a (tous)

.Supprimer un conteneur inactif et Supprimer de force: 

         docker rm [name conteneur]
         docker rm -f [name conteneur]

.Lister les images existants:

         docker image ls
         docker images

.Supprimer une image Docker:

         docker image rm [name image ou id ]
         docker rmi [name image ou id ]

.activer conteneur:

         docker start [id ou nom conteneur]
